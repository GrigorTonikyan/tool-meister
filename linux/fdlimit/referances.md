## Key Poetry Commands

### Project Setup

- `poetry new` - Create a new Poetry project
- `poetry init` - Create a new pyproject.toml file interactively

### Dependency Management

- `poetry add` - Add a package to your project
- `poetry remove` - Remove a package from your project
- `poetry show` - Show information about packages

### Environment Management

- `poetry env use` - Set specific Python version
- `poetry shell` - Activate the virtual environment

### Building and Publishing

- `poetry build` - Build source and wheel archives
- `poetry publish` - Publish the package to PyPI

### Development Workflow

- `poetry run` - Run a command in the virtual environment
- `poetry install` - Install the project dependencies
